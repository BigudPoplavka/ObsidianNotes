Этот метод генерирует объектный код фрагмента программы как только распознан синтаксис этого фрагмента.

Для него нужен набор подпрограмм, соотв-щих каждому правилу и каждой альтернативе в правилах грамматики.

Когда грамматический разбор даст результат в виде разпознаного фрагмента текста, который соотв-вует некоторому правилу исход. программы, тогда вызывается подпрограмма соотв-щая данному правилу. 

Название этих программ **Семантические**. Выполняют действия связанные со смыслом, который связан с соотв-щими конструкциями языка. Эти программы генерируют объектный код. Их второе название "программы генерации кода".
