#Параллелизм 
## Структура

**Структура типового MPI-приложения включает:**
1. инициализацию MPI, 
2. создание коммуникаторов, 
3. разделение задач между процессами, 
4. выполнение вычислений 
5. сбор результатов, 
6. завершение работы MPI 
7. завершение программы. 

**Обязательными элементами MPI-приложения являются**:
1. инициализация MPI с помощью функции `MPI_Init`, 
2. создание коммуникаторов с помощью функции` MPI_Comm_create`, 
3. передача данных между процессами с помощью функций `MPI_Send и MPI_Recv`
4. выполнение вычислений и сбор результатов с помощью функций `MPI_Reduce` и `MPI_Allreduce`, 
5. завершение работы MPI с помощью функции `MPI_Finalize`
6. завершение программы.
## Компиляция

**MPI**-приложения компилируются и запускаются с помощью **специальных компиляторов и запускателей**, которые обеспечивают **связь между процессами и передачу сообщений между ними**. Для компиляции MPI-приложения необходимо использовать **специальный компилятор**, который поддерживает **MPI**, например, **mpicc** для языка C или **mpif90** для языка Fortran. Для запуска MPI-приложения необходимо использовать запускатель, который обеспечивает запуск процессов и передачу сообщений между ними, например, mpirun или mpiexec.
