#Параллелизм 

**Футура** ( будущее ) представляет собой обертку, над каким-либо значением, вычисление или получение которого происходит от предписанного. Точнее, **Футура** предоставляет доступ к разделяемому состоянию , которое состоит из 2-х частей: **данных** (здесь лежит значение) и **флажок доступности**. 

`public async Task MethodWithReturnAsync()
``{
``    var content = await GetUrlAsStringAsync("http://google.com");
``}`

Под **future** обычно имеется в виду представление (**доступное только для чтения**) под **обещание** — изменяемый контейнер с однозначным освоением . **Будущее** может быть определено без указания того, из какого **обещания** будет получено значение.

- ## В .NET

`Task<T>` - это **future** для `Task` - источника возвращаемого значения
`TaskCompletionSource<T>` - это обещание **promise**
