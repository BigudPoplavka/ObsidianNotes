#Параллелизм 

**Процесс** — это экземпляр компьютерной программы, который выполняется. Он содержит код программы и ее текущую активность. В зависимости от операционной системы процесс может состоять из нескольких потоков выполнения, выполняющих инструкции одновременно.

Процесс обычно содержит следующие ресурсы:

• Образ исполняемого машинного кода, связанного с программой

памяти, в которую входят:

- исполняемый код
- специфические для процесса данные (входные и выходные)
- стек вызовов, который отслеживает активные подпрограммы и/или другие события
- куча, которая содержит промежуточные данные вычислений во время выполнения
- Дескрипторы ресурсов операционной системы, выделенные процессу, такие как файловые дескрипторы ( unix / linux ) и дескрипторы ОС (windows), источники данных и приемники
- Атрибуты безопасности (владелец процесса и набор разрешений, например разрешенные операции)
- Состояние процессора (контекст), такое как регистры и адресация физической памяти
