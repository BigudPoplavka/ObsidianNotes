
Абстрактный класс позволяет реализовать описание некой абстракции, включив в нее какую-то логику и состояние, для того чтобы передать его наследникам. Его применяют для выделения семейства классов с общим состоянием и логикой.

- может содержать логику
- может содержать состояние
- члены могут иметь разные модификаторы доступа
- не обязывает переопределять все абстрактные методы
